LWPR Library

This is a highly optimized implementation of Locally Weighted Projection Regression.

Vijayakumar S, D'Souza A, Schaal S (2005) Incremental online learning in high dimensions. Neural Computation 17: 2602-2634

This implementation has a lot of extra functionality for adaptive control and multiple forward-inverse models. Much more documentation of all of this is needed, including some clean-up of left-over experimental code.
